SECTION 04 · CORE

Gini 불순도 — 얼마나 섞였나?

한 그룹에 식용/독성이 얼마나 섞여있나. 0 = 순수, 0.5 = 반반(최악).

공식 Gini = 1 − Σ pᵢ²

트리 구조 출력
def gini(data): counts = {} for sample in data: lab = sample["label"] counts[lab] = counts.get(lab, 0) + 1 total = len(data) impurity = 1.0 for lab in counts: p = counts[lab] / total # 각 라벨 비율 impurity = impurity - p * p # 1 - Σp² 누적 return impurity

    SECTION 08 · LIMITS & WRAP-UP

    한계 · 그리고 정리

    의사결정트리도 만능은 아니다.

    ⚠️ 과적합 (Overfitting)
    깊이를 무한히 늘리면 train은 100%지만 보통 데이터는 test가 꺾인다 (외우기 시작). 버섯 데이터는 특징과 독성의 연관이 워낙 강하고 깨끗해서, 깊이 5에서도 test 99.8% 유지 → 과적합이 약하게만 나타남. "우리 데이터가 운이 좋은 케이스"임을 인지.
    📐 연속형 처리 부족
    우리 구현은 == 비교만 지원 (범주형 전용). 키·몸무게 같은 연속형은 <= 분기로 확장 필요.
    🌀 불안정성
    학습 데이터를 조금만 바꿔도 트리 구조가 크게 변함. → 실무는 여러 트리를 평균(Random Forest)으로 보완.
    👀 욕심쟁이(Greedy)
    매 분기마다 그 시점의 최선만 봄. 전체 관점에서 더 좋은 트리는 보장 X.
    사람이 손으로 짜던 if문을, 데이터가 자동으로 학습하는 모델.
