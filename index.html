SECTION 03 · PRINCIPLE

트리 = 분기 + 재귀

내부 노드와 잎 노드의 차이가 핵심입니다.

01
질문으로 두 그룹을 분리
02
각 그룹에서 또 질문 = 재귀
03
예측은 잎 노드에서만 (내부 노드는 질문만)
⏱ 50s
SECTION 04 · CORE

Gini 불순도 — 얼마나 섞였나?

한 그룹에 식용/독성이 얼마나 섞여있나. 0 = 순수, 0.5 = 반반(최악).

공식 Gini = 1 − Σ pᵢ²

☠️ 독성
poisonous

1 − 0.500² − 0.500² = 0.500

① gini() — 노드의 불순도
def gini(data):          데이터 묶음 → 불순도 한 숫자
    counts = {}          라벨별 개수 셀 통
    for s in data:      샘플 순회
        counts[s["label"]] += 1
    p = 1.0            1 − Σp² 시작
    for lab in counts:
        p -= (counts[lab]/n)**2  ↗ 트리 모든 'gini=' 숫자
    return p

def build_tree(data, d, max_d):
  if gini(data)==0 or d>=max_d:
    node.prediction = majority(data)  ↗ 잎 4개
