SECTION 03 · PRINCIPLE

트리 = 분기 + 재귀

노드에 마우스를 올려보세요 — 분기 노드와 잎 노드의 차이가 핵심입니다.

01
질문으로 두 그룹을 분리
02
각 그룹에서 또 질문 = 재귀
03
예측은 잎 노드에서만 (분기 노드는 질문만)

SECTION 04 · CORE

Gini 불순도 — 얼마나 섞였나?

한 그룹에 식용/독성이 얼마나 섞여있나. 0 = 순수, 0.5 = 반반(최악).

공식 Gini = 1 − Σ pᵢ²

SECTION 05 · IMPLEMENTATION

우리 코드 ↔ 실제 트리

함수 6개 · 외부 라이브러리 0개. 코드 한 줄 한 줄이 왼쪽 트리의 어디를 만드는지 따라가 보세요.

📐 max_depth=2 트리 (실제 출력) 식용 우세 독성 우세
①# gini() — 한 노드의 불순도를 계산 def gini(data): counts = {} for sample in data: counts[sample["label"]] = counts.get(sample["label"], 0) + 1 impurity = 1.0 for lab in counts: p = counts[lab] / len(data) impurity -= p * p ← 트리 모든 노드의 gini = 이걸로 계산 return impurity
②# best_split() — 모든 (특징,값) 시도, gain 최대 선택 def best_split(data): for feature in FEATURE_NAMES: ← 22개 특징 다 시도 for value in set(s["features"][feature] for s in data): left, right = split(data, feature, value) gain = base_gini - weighted_gini(left, right) if gain > best_gain: best = (feature, value) return best ← 루트엔 odor=="n" 선택 (gain=0.31)
③# build_tree() — 재귀로 트리 생성 def build_tree(data, depth, max_depth): if gini(data) == 0.0 or depth >= max_depth: node.prediction = majority_label(data) return node ← 잎 노드 4개 (식용/독성) feature, value, _ = best_split(data) left, right = split(data, feature, value) node.left = build_tree(left, depth+1, max_depth) ← YES 가지 node.right = build_tree(right, depth+1, max_depth) ← NO 가지 return node ← 분기 노드 3개
④# predict() — 뿌리부터 잎까지 따라 내려가기 def predict(node, sample): if node.prediction is not None: return node.prediction ← 잎 도착! 예측 반환 if sample["features"][node.feature] == node.value: return predict(node.left, sample) ← YES 따라가기 else: return predict(node.right, sample) ← NO 따라가기
